Hi, I'm Ryan

I am a computer engineering researcher who specializes in applied machine learning and computational intelligence, including neural networks, reinforcement learning, and large language models.

My current research interests are centered around education. Many of my recent publications focus on using new technologies to support students and teachers.

Projects

The PING System

The PING (Personalized Instruction and Need-aware Gaming) system is a generalized recommender system I've spearheaded, designed for easy integration into educational software.

My work on this project has been self-driven, requiring research, Python programming, data analysis, and reinforcement learning algorithm development. The PING system is my current main research focus.

Read More

Gridlock

Gridlock is an educational game made in Unity. The game is designed to augment the computer engineering curriculum by reinforcing students' knowledge of logic design. See a summary of Gridlock here.

My contributions to this project include programming in C#, environment design, game system overhauls, and the integration of AI-driven personalized assistance.

Read More

SPARC

SPARC (Systematic Problem-solving and Algorithmic Reasoning for Children) is a game for middle school students that teaches early science concepts while boosting problem-solving and computational thinking skills through a mobile game.

I've consulted with our research team to design new game scenarios, independently developed several game modules using Unity and C#, and am assisting with the ongoing developments using large language models.
