Hi, I'm Ryan

I am a computer engineering researcher who specializes in applied machine learning and computational intelligence, including neural networks, reinforcement learning, and large language models.

My current research interests are centered around education. Many of my recent publications focus on using new technologies to support students and teachers through recommender systems and serious games.

Projects

The PING System

The PING (Personalized Instruction and Need-aware Gaming) system is a generalized recommender system I've spearheaded as part of NSF-funded research. PING is designed for easy integration into educational software.

My work on this project has been self-driven, requiring research, Python programming, data analysis, and reinforcement learning algorithm development. More recently, PING development has moved toward neuro-symbolic AI, leveraging generative AI alongside the reinforcement learning developments to improve student recommendations. The PING system is my current main research focus.

Gridlock

Gridlock is an educational game made in Unity as part of an NSF-funded project. The game is designed to augment the computer engineering curriculum by reinforcing students' knowledge of logic design. See a summary of Gridlock here.

My contributions to this project include programming in C#, environment design, game system overhauls, and the integration of AI-driven personalized assistance.

SPARC

SPARC (Systematic Problem-solving and Algorithmic Reasoning for Children) is an AI-enabled game for middle school students that teaches early science concepts while boosting problem-solving and computational thinking skills through a mobile game.

I've consulted with our research team to design new game scenarios, as well as independently developed several game modules using Unity and C#, requiring programming, modelling, and writing. I'm also assisting with ongoing developments funded by Google in using large language models for educational support.

WaterPAL/WaterTALK

WaterPAL and WaterTALK are both under the WaterWorks project. WaterWorks is an EPA-funded project designed to increase student interest in wastewater treatment careers through educational outreach, interactive experiments for teachers, and educational mobile games.

As part of this project, my responsibilities included game module development and project dissemination, mainly though conference presentations and collaborations with the Camden City school district in New Jersey.

Earth Trek

Earth Trek is an educational mobile game made in collaboration with Rowan University's civil engineering department. It teaches basic concepts in geotechnical engineering, and is designed for both outreach to potential students and for a "remote lab" experience, as students can do simplified lab experiments.

My responsibilities on this project have included consulting with the research team on design work. I also developed a few modules in Unity and C# for inclusion into the game, as well as testing and fixing bugs.

ARSL

ARSL (Augmented Reality using Structured Light) was a computer vision project centered around using structured light patterns to create virtual 3D models of objects.

In my undergraduate work, I was responsible for implementing an existing and already-published algorithm in C++, as well as performing testing and data collection with the system.

Memsat

Rowan University's own cube satellite (cubesat) designed and built for NASA's seventh round of cubesats. Memsat required a great level of teamwork and coordination to ensure both successful and timely development.

My responsibilities on the project included programming internal communications between the main control system and the communications system using the UART standard.
